# python-owen #

Библиотека для работы с приборами фирмы [Овен] по протоколам Modbus и ОВЕН

Поддерживаются:

|  Модель  |  Модель  |  Модель  |  Модель  |  Модель  |
| :------: | :------: | :------: | :------: | :------: |
| [ТРМ101] | [ТРМ151] | [ТРМ210] | [МВ210]  |   [СИ8]  |
| [ТРМ136] | [ТРМ200] | [ТРМ212] | [МУ210]  |  [СИ30]  |
| [ТРМ138] | [ТРМ201] | [ТРМ251] | [МК210]  |  [ПР103] |
| [ТРМ148] | [ТРМ202] | [2ТРМ1]  | [ТРМ10]  |          |

Возможна поддержка других моделей путем добавления настроек в папку `owen/device`

Не со всеми моделями проверена работа библиотеки

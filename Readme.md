## python-owen ##

Библиотека для работы с приборами фирмы [Овен] по протоколам Modbus и ОВЕН

Поддерживаются:

|  Модель  | Проверено |  Модель  | Проверено |
| :------: | :-------: | :------: | :-------: |
| [ТРМ101] |     -     | [ТРМ201] |     +     |
| [ТРМ136] |     -     | [ТРМ202] |     +     |
| [ТРМ138] |     -     | [ТРМ210] |     -     |
| [ТРМ151] |     -     | [ТРМ212] |     -     |
| [ТРМ200] |     -     | [ТРМ251] |     -     |

Все настройки моделей хранятся в файле `owen/device.py`
